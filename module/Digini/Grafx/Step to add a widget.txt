
To add a widget you can follow other widget as template.

-----------------------------------------------------------------

lib_class_widget.h

Add your class definition here

-----------------------------------------------------------------

Create a source file lib_widget_<xxx>.cpp
use any other class as template.

in your <Workspace><Project>\module\Digini\Grafx\src\widget

-----------------------------------------------------------------

widget_cfg.h

Add every iteration of the widget in the this file for 
the application.

-----------------------------------------------------------------

lib_grafx_typedef.h

Add the const struct data definition use by the widget here.

-----------------------------------------------------------------

lib_grafx_enum_x_macro.h

Add the X_MACRO in the <enum Widget_e> list.

-----------------------------------------------------------------

lib_widget_variable.h

Add the external variable definition here.

-----------------------------------------------------------------

lib_widget_variable.cpp

Add the X_MACRO variable creation here.

-----------------------------------------------------------------

lib_gui_task.cpp

Add your widget creation code in <CreateAllWidget()>.

-----------------------------------------------------------------
